Le code

— J'ai acquis mes connaissances en ThreeJS à l'aide de la formation de Bruno Simon, ThreeJS-Journey. Pour construire mon projet, j'ai utilisé son Bundler, préconfiguré avec Vite.

— J'ai réalisé toute la partie code de mon site sauf le fichier EventEmitter.js, fourni par Bruno Simon ici. Je me suis également aidé de la documentation de ThreeJS.

— Pour toutes les différentes animations de mon site, j'ai consulté la documentation de GSAP.
